# 🎙️ J.A.R.V.I.S. — Local AI Assistant & Digital Majordomo

J.A.R.V.I.S. — локальный цифровой дворецкий, обладающий скверным характером, британским акцентом и прямым доступом к управлению вашей системой.

Проект построен на базе Ollama, что позволяет ему работать полностью оффлайн, сохраняя вашу приватность и вычислительные ресурсы.

## 🧠 Особенности (Features)
- Personality Engine: Джарвис — сноб. Он не будет вежливо помогать вам деградировать. Если вы решите поиграть вместо работы, готовьтесь выслушать порцию элитарного сарказма.

- Hybrid Intent Detection: Система сочетает мощь LLM для общения и быстрые регулярные выражения для мгновенного управления медиа-плеером.

- Media Control Core: Прямое управление воспроизведением (Next/Prev, Volume, Play/Pause) через симуляцию медиа-клавиш.

- Rich UI: Красивый терминальный интерфейс с потоковой передачей ответов (streaming), панелями и Markdown-разметкой.

- Context Awareness: Помнит последние этапы диалога для поддержания осмысленной беседы.

## 🛠 Технологический стек
Core: Python 3.10+

- Brain: Ollama (Model: dolphin3:8b). 
    - Предпочтительнее именно эта модель, так как она лишена типичной «податливости» и лучше вживается в роль.

- Interface: Rich (Terminal Formatting)

- System Interaction: pynput (Keyboard Controller)

- CLI: argparse

🚀 Быстрый старт
1. Подготовка
Убедитесь, что у вас установлена и запущена Ollama.

2. Установка зависимостей

```sh
pip install -r requirements.txt
```

3. Запуск
Вы можете использовать Джарвиса как полноценного собеседника или как CLI-инструмент:

Режим чата:

```sh
python main.py chat
```

Управление музыкой из терминала:

```sh
python main.py next  # Переключить на следующий трек
python main.py prev  # Вернуться назад
```

## 🎭 Системная логика (System Prompt)
Джарвис следует строгим директивам:

1. Миссия важнее эмоций: Его приоритет — ваша продуктивность.

2. Никакого буквализма: Он интерпретирует ваши жалобы на усталость как повод для едкого замечания, а не команду к выключению.

3. Sophisticated Russian: Использует естественный, сложный русский язык без «машинного» привкуса.
